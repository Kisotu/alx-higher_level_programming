================================
How to Use 5-text_indentation.py
================================

Defines a text-indentation function ``text_indentation(text)``.

Usage
=====

Text is printed with two new lines after each of the characters ``.``, ``?``,
and ``:``.

::

    >>> text_indentation = __import__('5-text_indentation').text_indentation
    >>> text_indentation("Hello?")
    Hello?
    <BLANKLINE>

No spaces are printed at the beginning of a line.

::

    >>> text_indentation("   Hello World.")
    Hello World.
    <BLANKLINE>

::

    >>> text_indentation("          ")

Similarly, no spaces are printed at the end of each printed line.

::

    >>> text_indentation("Howdy.   ")
    Howdy.
    <BLANKLINE>

::

    >>> text_indentation("    Docstring is king.    Just Document.   ")
    Docstring is king.
    <BLANKLINE>
    Just Document.
    <BLANKLINE>

New lines are only printed for the characters ``.``, ``?``, and ``:`` -
text not ending with one of these characters is not ended with a new line.

::

    >>> text_indentation("No period, just no stopping")
    No period, just no stopping

New lines within a string are printed as normal.

::

    >>> text_indentation("Hey! Here goes Nothing:\nPrinted.")
    Hey! Here goes Nothing:
    <BLANKLINE>
    <BLANKLINE>
    Printed.
    <BLANKLINE>

::

    >>> text_indentation("\n\n\n 3 new lines.")
    <BLANKLINE>
    <BLANKLINE>
    <BLANKLINE>
    3 new lines.
    <BLANKLINE>

::
    >>> text_indentation("Complicated \n new line.")
    Complicated 
    new line.
    <BLANKLINE>

Invalid Text
============

The paramter ``text`` must be a string. Otherwise, a TypeError is raised.

::

    >>> text_indentation(56)
    Traceback (most recent call last):
    TypeError: text must be a string

::

    >>> text_indentation({"one": 7, "two": 9})
    Traceback (most recent call last):
    TypeError: text must be a string

::

    >>> text_indentation(None)
    Traceback (most recent call last):
    TypeError: text must be a string
